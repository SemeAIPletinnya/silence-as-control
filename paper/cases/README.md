# Boundary Cases (Paper Export)

This folder contains paper-packaged exports derived from tracked evidence under `reports/`.

- `borderline_maybe_short_regen.csv`
- `borderline_maybe_short_regen_summary.json`

Regenerate with:

```bash
python scripts/export_boundary_cases.py
```

These files are used for the paper's boundary-pocket subsection and are intentionally scoped to the MAYBE_SHORT_REGEN lane.
